# 💰 Personal Finance Manager

A web-based application to track income, expenses, and manage personal finances efficiently.

---

## 🚀 Features

- 👤 User Registration & Login
- 🔐 Secure Authentication System
- 💵 Add Income & Expenses
- 📊 Dashboard Overview
- 📂 Category Management
- 🔄 Reset Password Functionality
- 📄 Generate Reports (PDF)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Tools: XAMPP / WAMP
